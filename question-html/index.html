Question-1: What are the different way to select an element in dom?

Different ways to select an element in dom using id -- getElementById using class -- getElementsByClassName using tagname -- getElementsByTagName using queryselector -- querySelector, querySelectorAll

Question-2: What is the different between innerHTML, innerText and textContent?

innerHTML are used to get and set the html value of an element. innerText are used to get and set the text content of an element by ignoring hidden elements. textContent are used to get and set the text content of an element including all child node.

Question-3: What is event delegation in the dom?

Event delegation is a technique where I can add event listener only on parent and it will work for all the child element of the parent. The event listener utilizes event bubbling to handle events on dynamically added or existing child elements. Using event.target, I can identify the specific child where I want to perform the event

Question-4: What is event bubbling in the dom?

Event bubbling is a javascript concept.It happens when an element receives an event , and that event bubbles up to its parent and ancestor element until it go to the root element
